# 平衡輸送計算

磁場閉じ込め核融合では核融合反応を効率的[^1]に発生させるため、閉じ込めたプラズマの挙動を把握しておく必要があります。1 億度を超える[^2]事が想定されるプラズマ内では粒子は日常生活では考えられない速度で走り回ります。つまり、挙動を計算することで温度や密度の分布を算出し、核融合反応によって発生するエネルギーの値や時間変化などを求めます。

一般にプラズマの輸送計算を行う場合、主にコアとペデスタル、そしてスクレイプオフ層（SOL 領域とも ）に分けて行います。

[内部輸送障壁](/tokamak/itb)という閉じ込め改善現象によって、プラズマ内の輸送の挙動は異なります。
