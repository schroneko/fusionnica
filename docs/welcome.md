# Welcome to Fusionnica

世界各国でちらほらと注目を集めてきた核融合。

核融合は多種多様な技術が盛り込まれた最先端の技術（50 年前から言われ続けている？）であるため、専門的に研究をされている方の中でも全体像を把握する事はとても難しいと言われています。

Fusionnica では、核融合界の Britannica を目指し、横断的に、そしてエントリーから研究者の方にとってもお役に立てる知識をお伝えします。

Fusionnica は核融合を専門とする学生及び研究者、OB の方々の多大な協力を経て公開できております。皆様に心から感謝します。
