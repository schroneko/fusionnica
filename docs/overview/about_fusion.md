# 核融合とは？

核融合とは何か、その質問に答える事はなかなかに難しい。

ある人は「エネルギー問題を解決する革新的な技術」と。

ある人は「核と名のついた何かキケンなもの」と。

ある人は「水素爆弾に使われた技術」と。

ある人は「プラズマに水素を入れてエネルギーを取り出すもの」と。

ある人は「地球上に作る人工的な太陽」と。

これらは全て正確な表現ではなくとも正しいです。

## 核融合とは何であるか？

では、どんな言葉と一緒に使われるか。
「核融合」と一緒に使われる言葉としてどんな言葉が思いつくでしょうか。核融合〇〇もしくは〇〇核融合です。

これまで聞いたことのある方はおおよそ次の回答になるはずです。

- 核融合発電
- 核融合炉
- 核融合反応
- 核融合エネルギー
- 常温核融合
