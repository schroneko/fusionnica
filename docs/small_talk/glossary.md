# 核融合用語集

## なぜ用語集を作るのか

類似の用語集はインターネットの各所に見受けられますが、初学者からするとわからない言葉をわからない言葉で説明してるものが多いと感じています。

当該ページでは核融合用語に対するいわば Q&A を目指している。初学者が理解に躓きやすい用語やわざわざ説明する程ではないが暗黙の了解となっている用語について解説する。

したがって、もしわからない事がありましたら追記しますので、是非ご連絡ください。

一つのテーマになるようであれば専用ページを、用語の意味さえ理解できれば良い内容であればこの用語集に随時追加する予定です。

では、早速用語集をどうぞ。

### ファーストプラズマ

核融合実験炉において、建設が完了し、真空を引いてプラズマを点火する初めての動作の事もしくはそのプロセスのことをファーストプラズマという。実際のところ、対外的にプラズマ実験が開始されたとプレスリリースを打つために公表である事も多く、実際に一番初めのプラズマ点火ではない事もある。

### プラズマパラメータ

核融合実験炉におけるプラズマを決定する変数の代表値をまとめたものをいう。例えば、プラズマの主半径、副半径、トロイダル磁場、プラズマ電流、エネルギー増倍率などが含まれる。プラズマパラメータはローソン条件による制約や目標値などから決定される。
