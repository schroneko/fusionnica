# 核融合用語集

## なぜ用語集を作ったか

類似の用語集はインターネットの各所に見受けられるが、初学者からするとわからない言葉をわからない言葉で説明してるものが多いと感じている。

当該ページでは核融合用語に対するいわば Q&A を目指している。初学者が理解に躓きやすい用語やわざわざ説明する程ではないが暗黙の了解となっている用語について解説する。

したがって、今後も追記をしていくだけでなく、もしわからない事があって、Fusionnica で解決できない事があれば是非ご連絡ください。

一つのテーマになるのであれば専用ページ、用語の意味さえ理解できれば良い内容であればこの用語集に随時追加する予定です。

では、早速用語集をどうぞ。

### ファーストプラズマ

核融合実験炉において、建設が完了し、真空を引いてプラズマを点火する初めての動作の事もしくはそのプロセスのことをファーストプラズマという。実際のところ、対外的にプラズマ実験が開始されたとプレスリリースを打つために公表である事も多く、実際に一番初めのプラズマ点火ではない事もある。
