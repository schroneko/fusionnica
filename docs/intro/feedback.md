# もし文章の間違いに気付いたら

この Fusionnica は、人間に例えるなら生まれたばかりの赤子です。

どんな本でも些細な間違いを含んでいるように、また時の経過で意図せず間違いに変わってしまう事もあるように、どうしても誤字脱字や正確でない表現、間違いを見落としている可能性があります。

できるだけ間違いを減らせるように努力して参りますが、必ず間違いが存在すると認識した上で読まれる事をお勧めしています。もし、読んでいて間違いを発見したら、ぜひ報告をしていただけると幸いです。

また、些細な疑問や結果として勘違いだとわかる事さえも、議論を通してお互いの理解の向上になるはずです。気軽にお問い合わせてください。
