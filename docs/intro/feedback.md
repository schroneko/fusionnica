# もし文章の誤りに気付いたら

この Fusionnica は例えるなら、まだ赤子です。

できるだけ誤りを減らせるように努力して参ります。

しかし、どんな本でも些細な誤りを含んでいるように、また時の経過で意図せず誤りに変わる事もあるように、どうしても誤字脱字や正確でない表現、誤りを見落としている可能性があります。

そのため、必ず誤りが存在すると認識した上で読まれる事をお勧めしています。もし、読んでいて誤りを見つけたなら、ぜひ報告をしていただけると幸いです。

また、些細な疑問や勘違いさえも、議論を通してお互いの理解の向上になるはずです。そのため、気軽に問い合わせていただいて構いません。
