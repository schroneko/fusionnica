# 参考リンク集

- [よくわかる核融合炉のしくみ](http://www.aesj.or.jp/~fusion/aesjfnt/jp/publications/rensai1.html)

核融合を専門に学び始めたいと考えたときに最もおすすめできるものがこちらの資料です。大学生向けに平易な文章で解説されているため、講義と併用したり、核融合の全体像を抑えたり、輪読したりと様々な用途に活用できます。

- [ITER CONFERENCES](https://www.iter.org/Conferences)

開催予定もしくは過去に開催した学会や研究会などの情報についてまとめられています。

- [プラズマ・核融合学会誌 バックナンバー](http://www.jspf.or.jp/journal/bn.html)

過去に出版されたプラズマ・核融合学会誌がなんと全て一般公開されている！（あまりに古いものは閲覧できない）それだけでなく、検索機能までついている！何か新しい用語に出会った際は**Google 検索するのではなく**こちらのページで検索する事を推奨します。

- [Science Direct](https://www.sciencedirect.com/)

言わずとしれた Elsevier の論文検索サービス。一般でもアブストラクトを読む事ができるが、もし大学や研究機関のアカウントがあれば無制限アクセス可能となり論文が読み放題です。

- [EUROfusion](https://www.euro-fusion.org)
