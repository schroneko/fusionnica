# 関連書籍

核融合とインターネットで検索すると、ほとんどの場合が一般向けの解説記事（それもツッコミを入れたい文章が散見される）であり、少し内容に踏み込もうとすると解説論文や学会、企業ページがヒットする。

筆者が学部 4 年の頃はこれに大変手間取った。なぜなら、専門的な文書はわからない事がわからない言葉で説明されている一方、一般向けの記事はわかった気になるだけで説明できる程の理解の助けにならない。

そんな中で役に立ったものの一つが書籍である。エントリーから専門（といっても修士以降は論文を読む方が多いかと思うが）まで、筆者が読んだものを主として役に立ちそうな書籍を紹介する。

念の為お伝えしますが、筆者が言うには大学院は主に研究をする場所であるため、読書ばかりしてうつつを抜かないように。（無論それもまた一興）

また、書籍での学習も十分効果はありますが、あくまで論文を読んだり研究を進めたりする上での補完である事を頭に置きつつ学習を進められたい。核融合を全体的に勉強しているだけで 5 年はかかってしまいます。

:::info
おすすめ度を ☆☆☆☆★ から ★★★★★ としています。
:::
