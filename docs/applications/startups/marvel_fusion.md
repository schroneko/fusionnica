# Marvel Fusion

Marvel Fusion はドイツの核融合スタートアップ企業である。
2019 年に設立された企業で、核融合スタートアップの中では比較的新興企業に位置付けられる。

## About

2019 年に現 CEO である Moritz Linden 氏および Georg Korn 氏、Pasha Shabalin 氏らが共同創業者となり Marvel Fusion を設立。2021 年 8 月現在 30 名の従業員を抱える[^1]。

レーザーを用いた爆縮をもとに核融合反応を起こす慣性核融合方式を採用している。

## Features

Marvel Fusion の特徴として、核融合炉実現のために現在主流と考えられている DT 反応ではなく、陽子とホウ素の核融合反応（ p-B 反応）を利用する点が挙げられる。

重水素と三重水素を衝突させる DT 反応では、反応性生物に中性子が含まれ、これをリチウムと反応させ燃料として再利用したり、中性子を増倍させてより高効率にエネルギーを取り出す事が考えられている一方で、一定数の反応しない中性子を吸収し、放射化した炉材料の回収およびメンテナンスの工数が核融合炉に必要とされる。

一方で、p-B 反応を利用した核融合炉の場合、反応生成物に中性子が存在せず、ヘリウム原子核のみであるため、TAE Technologies と共によりクリーンな核融合炉の完成形であるとも言われる。無論メリットばかりではなく、完成形と言われる程難易度は高く、DT 反応と比較して核融合反応を起こす条件が著しく厳しく、また仮に核融合炉を発電用途に利用する場合のエネルギー取り出しの問題など課題が残っている。
